# Spring Security Basic

## 기술 스택
* IDE
```
 - IntelliJ IDEA
```
* Language
```
 - Java
```
* Back-end
```
 - Spring Boot
 - Spring Security
 - Maven
 - Lombok
```
* Database
```
 - MySQL
```
</br>

## 프로젝트 목적
* Spring Security 프레임 워크의 기본적인 작동원리 및 사용법을 터득한다.

* 일반 로그인 및 회원가입의 인증 과정을 파악하고 구현 방법을 터득한다.

* OAuth 로그인 및 회원가입의 인증 과정을 파악하고 구현 방법을 터득한다.
</br>


## 프로젝트 기능
* 일반 로그인 및 회원가입 기능

* 구글 로그인 및 회원가입 기능

* 페이스북 로그인 및 회원가입 기능

* 네이버 로그인 및 회원가입 기능
